# Battle-of-the-Ancient-Lands
# Java Battle Game (OOP + Swing)

A simple 2D turn-based battle game developed in Java using Object-Oriented Programming principles and Swing for the GUI.

## Features
- Two playable characters 
- Turn-based combat mechanics
- Game Over screen with replay option
- Simple battle log for in-game events

## Technologies Used
- Java
- Java Swing 
- OOP concepts 

## How to Run
1. Open the project in IntelliJ IDEA or any Java IDE
2. Run the main class which is in BattleGui
